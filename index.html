CSSslidy

No frameworks. No plugins. Just CSS and 1.5K of vanilla JavaScript.

  1. Ensure that all the images you wish to use are exactly the same size.
  2. Place images in a element with an id of slidy: 
    <div id="slidy-container">
  <figure id="slidy">
    <img src="eyes.jpg" alt>
    <img src="lou.jpg" alt>
    <img src="lucielle.jpg" alt>
    <img src="lucie.jpg" alt>
  </figure>
</div>
  3. Ensure that slidy is contained by an element that has overflow: hidden applied to it: 
    #slidy-container { width: 70%; overflow: hidden; }
  4. Add cssslidy.js to the bottom of the page.

Optional: adjust timing variables to adjust the pace of the animation (see source for more details)
